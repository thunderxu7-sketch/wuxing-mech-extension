如何安装 · How to install

  1. 下载最新发布版（zip 包） Download the latest release (zip)
  2. 解压到本地任意目录 Unzip into a local folder
  3. 打开 Chrome，访问 chrome://extensions Open Chrome and go to chrome://extensions
  4. 右上角开启「开发者模式」 Enable "Developer mode" (top-right)
  5. 点击「加载已解压的扩展程序」选择解压目录 Click "Load unpacked" and select the unzipped folder
  6. 在工具栏点开图标，输入出生时间即可 Click the toolbar icon and enter your birth time
